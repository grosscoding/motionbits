A component that animates a number counting up or down from one value to another. The animation uses spring physics for a natural, dynamic feel and supports custom separators for better readability of large numbers.

## Features

- Animate counting up or down between any two numbers
- Customizable animation duration and delay
- Spring physics for natural animation feel
- Optional thousands separator
- Supports both positive and negative numbers
- Intersection Observer integration for on-scroll animations

## Example

<ExampleWrapper id="count-up">
  <CountUp from={0} to={1000} duration={2} />
</ExampleWrapper>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `from` | number | `0` | Starting value for the animation |
| `to` | number | required | Ending value for the animation |
| `direction` | string | `'up'` | Direction of counting: 'up' or 'down' |
| `delay` | number | `0` | Delay before animation starts (in seconds) |
| `duration` | number | `2` | Duration of the animation (in seconds) |
| `separator` | string | `''` | Character to use as thousands separator |

## Accessibility

<Card title="Accessibility Considerations">
  - The component uses semantic HTML elements for better screen reader support
  - Animation duration can be adjusted for users who may need more time to process changing numbers
  - Consider using ARIA live regions for important number changes that need to be announced by screen readers
  - For users with cognitive disabilities, consider using slower animations and clear separators for large numbers
</Card>
