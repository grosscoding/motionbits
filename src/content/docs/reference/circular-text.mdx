A component that displays text in a circular pattern with continuous rotation animation. The text is arranged in a circle, with each letter positioned at equal intervals around the circumference. The animation can be modified on hover with different behaviors.

## Features

- Arranges text in a perfect circle
- Continuous rotation animation
- Interactive hover effects with multiple options
- Customizable animation speed
- Responsive design

## Example

<ExampleWrapper id="circular-text">
  <CircularText text="CIRCULAR•TEXT•ANIMATION•" onHover='slowDown'/>
</ExampleWrapper>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | string | `''` | The text to display in a circle |
| `spin-duration` | number | `20` | Duration of the spin animation in seconds |
| `on-hover` | string | `'speedUp'` | Behavior on hover: 'speedUp', 'slowDown', 'pause', 'goBonkers' |

## Accessibility

<Card title="Accessibility Considerations">
  - The circular text is decorative and may be difficult to read for some users
  - Consider providing an alternative text representation for screen readers
  - Ensure sufficient color contrast between the text and background
  - Users with motion sensitivity may prefer reduced motion - consider adding a media query to respect user preferences: `@media (prefers-reduced-motion: reduce)`
</Card>
