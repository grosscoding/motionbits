The Text Cursor component creates an interactive trail of icons or emojis that follow the user's cursor movement. This creates a playful and engaging interaction that can add personality to your website.

## Features

- Customizable icon or emoji
- Adjustable trail length and spacing
- Configurable animation speed
- Smooth cursor following with physics-based animation
- Automatically cleans up when cursor leaves the component area

## Example

<TextCursor text="⚛️" />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | string | `'⚛️'` | The icon or emoji to display in the trail |
| `delay` | number | `0.01` | Animation delay between trail elements |
| `spacing` | number | `100` | Spacing between trail elements in pixels |
| `class` | string | `''` | Additional CSS classes |
