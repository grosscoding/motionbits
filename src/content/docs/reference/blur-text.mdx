The Blur Text component creates an engaging text reveal animation with a blur effect. Text elements (words or individual letters) appear with a smooth transition from blurred to sharp, creating a dynamic and attention-grabbing effect.

## Features

- Animate text by words or individual letters
- Configurable animation direction (top or bottom)
- Intersection Observer integration for scroll-triggered animations
- Customizable animation timing and easing

## Example

<BlurText text="This text animates with a blur effect" />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | string | `''` | The text to animate |
| `animateBy` | string | `'words'` | Animation method: `'words'` or `'letters'` |
| `direction` | string | `'top'` | Direction of animation: `'top'` or `'bottom'` |
| `delay` | number | `50` | Delay between each word/letter animation in milliseconds |
| `class` | string | `''` | Additional CSS classes |
