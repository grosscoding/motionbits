A visually engaging component that creates a grid of pixels that respond to mouse movement, leaving a trail effect. The component uses WebGL and Three.js to render a customizable pixel grid that follows your cursor with smooth animations.

## Features

- Interactive pixel grid that follows mouse movement
- Customizable grid size, trail size, and animation duration
- Optional gooey filter effect for smoother, more fluid animations
- Customizable color for the pixel trail
- Works with both mouse and touch events
- Optimized WebGL rendering for smooth performance

## Example

<PixelTrail gridSize={50} trailSize={0.1} maxAge={250} interpolate={5} color="#00d8ff" gooeyFilter={true} gooeyFilterId="goo-filter" gooeyFilterStrength={2}/>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `grid-size` | number | `40` | Size of the pixel grid (higher values create smaller pixels) |
| `trail-size` | number | `0.1` | Size of the trail effect (0.0 to 1.0) |
| `max-age` | number | `250` | Duration in milliseconds before trail points fade out |
| `interpolate` | number | `5` | Smoothness of the trail animation (higher values create smoother trails) |
| `color` | string | `'#ffffff'` | Color of the pixel trail |
| `gooey-filter` | boolean | `false` | Whether to apply a gooey filter effect to the trail |
| `gooey-filter-id` | string | `'goo-filter'` | ID for the SVG filter (useful when multiple instances exist) |
| `gooey-filter-strength` | number | `10` | Strength of the gooey filter effect |
